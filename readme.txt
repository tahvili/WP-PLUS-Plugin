== Description ==

This design was made by kiuloper.com, however the base code for this theme comes from the admin theme Aquila. The purpose of this template is not for commercial use at all. It is absolutely free to use.


== Installation ==

**Via FTP**

1. Upload **wp_plus-admin-theme** to the /wp-content/plugins/ directory.
2. Activate **WP Plus Admin Theme** through the 'Plugins' menu in WordPress.
3. That's it! The default settings are applied automatically, and you can hide other pages in the options page at **Settings > Admin Template** if you wish.
